O Gatitude é um projeto pensado para ONGs de animais que, muitas vezes, ainda precisam lidar com processos totalmente manuais por não terem estrutura suficiente para organizar todas as informações do dia a dia. A ideia surgiu a partir dessa dificuldade comum, já que muitas dessas organizações fazem um trabalho incrível, mas acabam tendo pouco controle sobre dados importantes por falta de ferramentas adequadas.
O projeto tem como proposta a criação de um site que ajude a centralizar e organizar as informações dos animais, começando pela apresentação dos animais disponíveis para adoção, mostrando dados como idade, porte e outras informações relevantes. Além disso, o sistema também prevê um controle básico de vacinas e cuidados, facilitando o acompanhamento da saúde dos animais.
Outra parte importante do projeto é o controle de ração, pensado para ajudar na organização da alimentação de acordo com a idade e o tamanho dos animais, trazendo mais previsibilidade e evitando desperdícios. Com isso, a ideia é tornar a gestão mais simples, prática e acessível, mesmo para ONGs que não possuem uma grande estrutura tecnológica.
O Gatitude funciona como um projeto de estudo e prática.
